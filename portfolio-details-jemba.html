Jemba bird is a game application for android smartphones that takes the arcade genre with a similar application, namely Flappy bird

Jemba Bird can help get rid of fatigue and boredom after work and can train the child's physical motor skills

This game application is made using the Apache Cordova hybrid system apps with NPM (Node Package Manager) and the programming language for the frontend, namely html5, css and javascript.

while for the backend using php with mysql database
