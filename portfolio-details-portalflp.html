Project information

  • Category: Web Base Application
  • Client: Portal Front Line People
  • Project date: 22 June, 2018
  • Project URL: http://portal.daya-wisesa.com:8081/flp (Internal Consumtion)

FLP Web Portal (Front Line People) is a portal for the identity and achievement of each Front Line Sales Marketing in a company.

This system can track the extent of sales achievements per month, even per day.

Another feature of this system is the Product Knowledge Training Report Card and being able to see the product stock at each sales outlet.

This system is built using web-based programming (HTML, CSS, JAVASCRIPT) and PHP and MYSQL for backend and database programming
