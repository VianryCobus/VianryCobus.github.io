Project information

  • Category: e-commerce mobile apps
  • Client: Startup Tole Manado
  • Project date: 01 March, 2020
  • Project URL: https://play.google.com/store/apps/details?id=com.tolemdo.tole

the tole application is an e-commerce application created by a Manado startup, namely TOLe, so that it can help tourists or people who want to shop for souvenirs or kitchen ingredients only with a smartphone can be accessed anywhere and anytime

The tole mobile application is built using the HTML 5 programming language and Javascript for the frontend, while PHP for the backend and MYSQL for database and uses the hybrid apps method known as Apache cordova with NPM (Node Package Manager).
